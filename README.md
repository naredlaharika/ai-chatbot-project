# ai-chatbot-project
Ai chatbot that answers user queries using OpenAI API built using Python 

## Features
- Takes user input
- Generates AI responses
- Interactive chatbot

## Technologies Used
- Python
- OpenAI API

## How to Run
1. Install Python
2. Install openai library
3. Add your API key in the code
4. Run the program
